# Merry360x Mobile Design Preview

This folder contains a quick visual preview of the app design direction (not production code).

## Included mockups
- Home / Discovery
- Listing Detail
- Booking Confirmation

## Design tokens
- Coral: `#E2555A` (primary actions)
- White: `#FFFFFF` (background)
- Gray: `#F5F5F5` (cards/sections)
- Rounded controls + soft shadows

## How to preview
Open `index.html` directly in your browser.

Or run:

```bash
cd merry360x_mobile_design_preview
python3 -m http.server 5050
```

Then open http://localhost:5050
